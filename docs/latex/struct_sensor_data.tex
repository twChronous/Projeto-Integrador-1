\doxysubsection{Detailed Description}
Estrutura consolidada de dados de sensores. 

Combina dados do acelerômetro, altímetro e timestamp em uma única estrutura para transmissão e processamento.

Uso de \#pragma pack para garantir alinhamento de bytes consistente entre diferentes plataformas
